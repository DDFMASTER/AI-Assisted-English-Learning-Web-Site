使用vue3完成前端的设计。前端的样式参考htmlprototype中的html文件，结合guidance中md文件和src中已有的后端代码，为动态网页中相关按钮绑定相关触发器，便于接下来的功能实现。

# 整体页面关系与路由规划

| 页面名称 | 路由路径 | 功能描述 |
|---------|---------|---------|
| materials.html | /materials | 智能读物匹配主页 |
| assessment.html | /assessment | 自适应测评（含开始/答题两个内部状态） |
| result.html | /result | 测评结果报告 |
| profile.html | /profile | 个人中心与设置 |
| reader.html | /reader | AI 沉浸式阅读器（独立页面，通常从读物卡片进入） |
| LoginPage | /login | 用户登录与注册 |
| AdminPage | /admin | 管理后台 |
| EnglishWorldPage | /english-world | 英语天地（情景对话、真人PK、单词接龙） |

---
首先进入网页的首页参考materials.html文件，在未登录状态下，进行任何操作都会弹出登录窗口，登录窗口中又包括注册按钮，也可以点击右上角的登录按钮进行登录注册。

### 2. 左侧内容区

#### 2.1 Tab 切换按钮

| 元素 | 作用 | Vue 3 实现 |

| 进阶类 (期刊/原著) | 切换到"进阶类"文章列表 | 用 `v-model` 绑定当前激活的 tab 值，动态显示不同数据 |

#### 2.2 文章卡片

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 整张文章卡片 | 点击后进入阅读器 reader.html | 最外层 div 绑定 `@click="goToReader(articleId)"`，跳转 `/reader?id=xxx` |

## 二、assessment.html — 自适应测评页

已在之前的回答中详细分析，这里我们只提炼关键点并补充 Vue 3 实现细节。

### 状态 1：开始屏幕

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 开始测评 按钮 | 进入答题模式 | `@click="startAssessment()"` → 修改 `currentScreen = 'quiz'`，并向后端请求试卷 |

### 状态 2：答题屏幕

#### 顶部

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 退出测评 按钮 | 中断测评 | 点击后弹窗确认，确认后重置状态并跳转 `/materials` |
| 题目序号 & 进度条 | 实时反映答题进度 | 使用计算属性 `currentIndex` 控制 |

#### 答题区

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 选项卡片 (A/B/C/D) | 选择答案 | `@click="selectOption(optionId)"`，用 `selectedOption` 控制高亮 |
| 理解程度标记：看得懂/模糊/看不懂 | 记录用户对文章的理解自我评价 | 三个按钮，绑定 `comprehensionLevel`，用 `:class` 动态激活样式 |

#### 底部导航

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 上一题 按钮 | 回到前一题 | `@click="prevQuestion"`，第一题时 `disabled` |
| 下一题 按钮 | 前往下一题 / 提交测评 | `@click="nextQuestion"`；最后一题时按钮文案可变为"查看结果"，点击后提交答案并跳转到 `/result` |
| 自动保存进度 | 提示文字，非按钮 | 使用 `watch` 监听答案变化，自动调用保存 API |

## 五、reader.html — AI 沉浸式阅读器

这是交互最丰富的页面，包含多个悬浮功能。

### 1. 沉浸式导航栏

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 返回按钮 (←) | 返回读物匹配页 | `<router-link to="/materials">` |
| 文章标题 | 显示当前文章，非按钮 | 动态显示 |
| 读物匹配/任务管理等 链接 | 同全局导航 | 可复用 AppNav 的部分，但此处是简化版 |
| 对照翻译 按钮 | 切换显示/隐藏译文 | 绑定 `toggleTranslation()`，通过 CSS 或条件渲染显示译文 |
| 字体大小按钮 (T) / 书签按钮 | 调整字号 / 添加书签 | `@click` 调整 `fontSize` 变量；`@click` 调用书签 API |
| 点赞按钮 (爱心) | 点赞/取消点赞文章 | `ph:heart-bold`（空心）或 `ph:heart-fill`（实心红色，已点赞），位于书签按钮旁边。`@click` 调用 `POST /api/article/like`，状态通过 localStorage（`aael_liked_<articleId>`）持久化。hover 显示 tooltip 和当前点赞数。支持暗色模式和移动端。 |

### 2. 文章内容区

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 可交互单词 (interactive-word) | 点击弹出单词释义浮窗 | `@click="showWordPopover(wordData)"`，根据点击位置计算浮窗坐标，并显示单词、音标、释义等 |
| 文化注解标记 (★) | 点击弹出文化背景浮窗 | `@click="showCultureNote(data)"`，同样用绝对定位浮窗 |

**重点**：浮窗需要动态定位，且点击页面其他地方需要关闭。可用 Vue 的自定义指令或全局点击监听实现。

### 3. 苏格拉底式提问卡片

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| "写下你的想法" 按钮 | 打开回答输入框 | `@click="showAnswerInput = true"`，显示输入区域 |
| "稍后回答" 按钮 | 关闭提问卡片或标注为待回答 | 可将问题加入待回答列表 |

### 4. 底部固定操作栏

| 元素 | 作用 | Vue 3 实现 |
|------|------|-----------|
| 进度条 + 百分比 | 展示阅读进度，非按钮 | 根据滚动位置或章节计算出百分比 |
| 分段翻译 按钮 | 按段落显示翻译 | `@click="toggleTranslation()"` 或切换翻译模式 |
| 语音朗读 按钮 | 开始/暂停 TTS 朗读 | 调用浏览器 `SpeechSynthesis` API |
| 目录 按钮 | 打开文章大纲/章节跳转 | `@click="showToc = true"`，弹出侧边栏或下拉菜单 |
| 设置 按钮 | 调整阅读器设置（字体、主题等） | 打开设置面板 |
| 下一段 按钮 | 滚动到下一个段落 | `@click="scrollToNextParagraph()"` |

### 5. 两个浮窗 (单词浮窗、文化注解浮窗)

这些不是按钮，但由按钮触发，且浮窗内有交互：

| 浮窗内元素 | 作用 | Vue 3 实现 |
|-----------|------|-----------|
| 加号按钮 (➕) / 加入生词本 | 将单词加入生词本 | `@click="addToVocab(word)"` |
| 查看详细解析与例句 | 跳转或弹出更详细的单词页面 | 可打开新浮窗或跳转 |
| 文化浮窗 | 仅展示，无按钮 | - |

### 2. 全局状态管理 (Pinia)

建议创建以下 Store：

- **useUserStore**：用户信息、等级、经验、设置项
- **useAssessmentStore**：测评状态、题目、答案、进度
- **useReaderStore**：当前阅读文章、进度、书签、生词操作
- **useTaskStore**：本日/本周任务列表、完成状态

### 3. 路由设计

```javascript
const routes = [
  { path: '/', redirect: '/materials' },
  { path: '/materials', component: MaterialsPage },
  { path: '/assessment', component: AssessmentPage },
  { path: '/result', component: ResultPage },
  { path: '/profile', component: ProfilePage },
  { path: '/reader', component: ReaderPage },
  { path: '/login', component: LoginPage },
  { path: '/admin', component: AdminPage },
  { path: '/english-world', component: EnglishWorldPage },
]
```

### 4. 交互细节注意事项

- **浮窗定位**：可使用 `teleport` 挂载到 `body`，监听 `click outside` 关闭
- **图表**：ECharts 在 `onMounted` 初始化，需在 `onUnmounted` 销毁实例，窗口 `resize` 时重绘
- **保存草稿**：答题页和阅读器进度需要利用 `watch` 和 `beforeRouteLeave` 自动保存
